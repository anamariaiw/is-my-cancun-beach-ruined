# Is My Cancun Beach Ruined?

A tiny Next.js app that asks OpenAI to do live web research about Cancun/Riviera Maya seaweed risk and returns a quick trip backup plan.

## Important

Do not paste your OpenAI API key into the website code.

Add it in Vercel as:

OPENAI_API_KEY

## Deploy

1. Upload this project to GitHub.
2. Import the GitHub repo into Vercel.
3. In Vercel, open Project Settings > Environment Variables.
4. Add OPENAI_API_KEY with your OpenAI API key.
5. Redeploy.
